# 运行系统

  1. 下载源码
git clone https://gitee.com/zhaoeryu/eu-vue3.git
  1. 安装项目依赖

进入到项目的根目录，执行以下命令：

npm i
  1. 启动系统
npm run dev

当你看到以下日志时，说明系统已经启动成功了。

  VITE v4.4.9  ready in 607 ms

  ➜  Local:   http://127.0.0.1:8080/
  ➜  Network: use --host to expose
  ➜  press h to show help
